# 宗翰與若慈婚禮請帖網頁 🎉

一個精美的三層互動式婚禮請帖網頁，包含動畫效果和線上出席確認表單。

## 📋 功能特性

### 第一層：信封頁面
- 💌 精美的婚禮信封設計
- ❤️ 飛舞的愛心動畫
- 📝 婚禮日期標題
- 🎯 點擊信封進入下一層

### 第二層：故事線頁面
- 📸 分段式故事展示
  - 相識：初次相遇的故事
  - 交往：甜蜜回憶的分享
  - 結婚：步入婚姻的承諾
- 💬 優美的文字敘述
- 📬 邀請訊息和 RSVP 按鈕

### 第三層：感謝與表單頁面
- 🙏 感謝蒞臨的訊息
- 📍 宴客資訊卡片
  - 地點、日期、時間
- 🗺️ 位置地圖位置
- 🅿️ 停車位置說明
- 📋 線上出席確認表單
  - 姓名欄位
  - 出席人數選擇
  - 出席確認選項
  - 備註欄位

## 🎬 過場動畫

- **信封開啟動畫**：信封蓋子打開，照片飛出
- **淡出轉換**：平滑的層級切換效果
- **手寫筆記動畫**：填寫表單前的過場動畫

## 🚀 使用方法

1. **開啟網頁**
   - 直接在瀏覽器中打開 `index.html`

2. **流程操作**
   - 第一層：點擊信封查看動畫
   - 第二層：向下滾動瀏覽故事，點擊「填寫表單」
   - 第三層：填寫出席確認表單

3. **自訂內容**
   - 編輯 `index.html` 修改文字內容
   - 編輯 `styles.css` 修改顏色和樣式
   - 編輯 `script.js` 修改互動邏輯

## 🎨 自訂選項

### 修改顏色主題
在 `styles.css` 中找到漸層色設定：
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

### 修改文本內容
在 `index.html` 中編輯：
- 標題：`<div class="title-text">宗翰與若慈婚禮<br>2026.10.25</div>`
- 故事文本：編輯 `.section-content` 內的內容
- 宴客資訊：編輯 `.info-detail` 區域

### 新增照片
將照片替換為真實圖片：
```html
<div class="placeholder-image">📸 相識的照片</div>
```
改為：
```html
<img src="images/meet.jpg" alt="相識">
```

## 🎯 主要動畫

| 動畫名稱 | 效果 |
|--------|------|
| `floatUp` | 愛心飄升動畫 |
| `flapOpen` | 信封打開動畫 |
| `fadeInDown` | 標題淡入下滑 |
| `popIn` | 元素彈跳出現 |
| `slideUp` | 向上滑動進入 |
| `notebookSlideIn` | 筆記本滑入 |
| `handWriteAnimation` | 手寫動畫 |

## 💡 提示

- 第一次訪問時，建議用全屏模式體驗最佳效果
- 音樂背景：可在 `index.html` 中 `<body>` 標籤內添加背景音樂
  ```html
  <audio autoplay loop>
      <source src="music.mp3" type="audio/mpeg">
  </audio>
  ```

- 表單數據：目前會顯示成功訊息，可連接後端 API 保存數據
